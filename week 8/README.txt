Trong bài tập tuần này, chúng ta sẽ học cách xây dựng mô hình mạng nơ-ron dựa trên thư viện Keras.


1. Cài đặt Keras

$ pip install keras

2. Dữ liệu
Để sử dụng được bộ dữ liệu ảnh CIFAR10 của Keras, chúng ta cần download về trong lần chạy đầu tiên.
Mỏ file "Neural Network.ipynb" và chạy phần code "Import dữ liệu" tại nhà trước khi đến lớp để download dữ liệu về.
